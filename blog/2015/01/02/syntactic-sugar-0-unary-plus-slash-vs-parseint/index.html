By placing a ‘+’ infront of a single integer the ‘+’ method triggers its unary functionality which is actually to convert the string into its integer equivalent. Additionally, you can use the ‘-’ for negative numbers.

This functionality does differ a bit from parseInt in that it only works for decimal numbers while parseInt can convert to other number bases.
